# Система учета показаний счетчиков ЖКХ (MeterReader)

Система для учета и анализа показаний приборов учета коммунальных услуг.

## Технологии
- Python 3.8+
- PyQt6
- SQLite
- pandas, matplotlib, openpyxl

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```

## Первый запуск

При первом запуске создается база данных с администратором:
- Логин: `admin`
- Пароль: `admin`

## Карта города

Для работы карты города администратора:
1. (Опционально) Создайте пример карты: `python create_map.py`
2. Или поместите свое изображение карты в файл `city_map.png` в корне проекта
3. При добавлении здания укажите координаты на карте (X, Y, ширина, высота) в диапазоне 0-1000
4. При клике на здание на карте откроется диалог с информацией о счетчиках, показаниях и пользователях

## Особенности
- Интерактивная карта города для администратора
- Учет объектов недвижимости и счетчиков
- Ввод показаний с валидацией (проверка на уменьшение показаний)
- Автоматические расчеты расходов и сумм к оплате
- Отчеты и графики потребления
- Напоминания о сроках передачи показаний и поверки счетчиков
- Экспорт отчетов в Excel
