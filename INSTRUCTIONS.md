# Инструкция по использованию системы учета показаний счетчиков ЖКХ

## Быстрый старт

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. (Опционально) Создайте пример карты города:
```bash
python create_sample_map.py
```

3. Запустите приложение:
```bash
python main.py
```

4. Войдите в систему:
   - Логин: `admin`
   - Пароль: `admin`

## Работа администратора

### Карта города

1. При первом запуске карта будет пустой (или покажет пример, если создали через `create_sample_map.py`)
2. Для добавления здания:
   - Нажмите "Добавить здание"
   - Заполните данные объекта
   - Укажите координаты на карте (X, Y, ширина, высота) в диапазоне 0-1000
   - Здание появится на карте красным прямоугольником

3. Для работы с зданием:
   - Кликните на здание на карте
   - Откроется диалог с вкладками:
     - **Счетчики** - список всех счетчиков объекта
     - **Показания** - история показаний
     - **Пользователи** - список пользователей системы

### Добавление счетчика

1. Откройте здание на карте
2. Нажмите "Добавить счетчик"
3. Заполните данные:
   - Тип счетчика (холодная/горячая вода, электроэнергия, газ, отопление)
   - Серийный номер
   - Место установки
   - Тариф
   - Единица измерения
   - Даты установки и поверки

### Ввод показаний

1. Откройте здание на карте
2. Нажмите "Ввести показания"
3. Если счетчиков несколько, выберите нужный
4. Введите дату и показание
5. (Опционально) Прикрепите фото счетчика
6. Система автоматически проверит, что показание не меньше предыдущего

## Работа пользователя

### Вкладки интерфейса

1. **Объекты** - список всех объектов недвижимости
2. **Счетчики** - список всех счетчиков
3. **Показания** - история всех показаний с расчетами
4. **Отчеты** - графики и экспорт данных

### Ввод показаний

1. Перейдите на вкладку "Показания"
2. Нажмите "Ввести показания"
3. Выберите счетчик (если их несколько)
4. Заполните форму и сохраните

### Экспорт отчетов

1. Перейдите на вкладку "Отчеты"
2. Нажмите "Экспорт в Excel"
3. Выберите место сохранения файла

## Уведомления

Система автоматически проверяет:
- Сроки поверки счетчиков (за 30 дней до даты)
- Просроченные показания (если не переданы за текущий месяц)

Уведомления отображаются в верхней части интерфейса администратора.

## Координаты на карте

Координаты указываются в относительных единицах (0-1000):
- X, Y - позиция левого верхнего угла здания
- Ширина, Высота - размеры здания на карте

Система автоматически масштабирует координаты под размер окна карты.

## Резервное копирование

База данных автоматически создается при первом запуске. Для резервного копирования используйте метод `backup_database()` класса `Database`.
